# 🦸‍♂️ 캐릭터 설정 및 디자인 명세서 (Character Design & Data Specification)

본 문서는 타워 디펜스 게임에 등장하는 10종의 캐릭터 객체(Entity)와 UI/UX 렌더링을 위한 시각적(Visual) 데이터를 정의합니다.

**[AI Agent 지침]** 1. **Data Structure:** 본 문서의 데이터를 바탕으로 캐릭터의 기본 정보, 스킬 트리, 시각적 속성을 포함하는 JSON 파일 또는 Kotlin `enum class` / `sealed class` 를 생성할 것.
2. **Visual Implementation:** `MainColor`와 `Visual Concept`은 안드로이드 Canvas 또는 UI 컴포넌트를 그릴 때 기본 테마 색상 및 파티클 이펙트 색상으로 활용할 것.
3. **Evolution State:** 캐릭터의 상태(`evolutionGrade`)가 변경될 때마다 `Skill`과 `Visual Change` 속성이 동적으로 렌더링 및 적용되도록 MVC/MVVM 패턴으로 분리할 것.
