# 👾 몬스터 밸런스 시트 (Monster Balance Sheet)

본 문서는 타워 디펜스 게임의 라운드별 난이도 곡선과 몬스터 스펙을 관리하기 위한 설계 가이드입니다.
**[AI Agent 지침]** 1. 모든 체력(HP), 방어력, 수량 데이터는 정수형(Integer)으로 처리하며, 부동소수점 연산 결과는 내림(Floor) 처리할 것.
2. 경제(Gold) 및 재화 관련 로직은 이 문서가 아닌 `economy_balance_sheet` 문서를 최우선으로 따를 것.

## 1. 10라운드 단위 스테이지 명세 (테마 및 보스)

**[AI Agent 지침]** * `currentRound % 10 != 0`: 일반 라운드 진행 (다수의 일반 몬스터 스폰)
* `currentRound % 10 == 0`: 보스 라운드 진행 (해당 라운드는 일반 몬스터 없이 단 1기의 보스 몬스터만 스폰)

| 구간 | 맵 테마 | 몬스터 특징 | 보스 (10R 단위) | 보스 특수 기믹 (구현 목표) |
| :--- | :--- | :--- | :--- | :--- |
| **1~10** | 평온한 숲 | 표준형 (튜토리얼) | 거대 나무정령 | 높은 체력, 느린 이동속도 |
| **11~20** | 버려진 광산 | 방어형 (물리 저항) | 강철 골렘 | 기본 방어력에 +20 추가 가중치 |
| **21~30** | 건조한 사막 | 속도형 (빠른 이동) | 모래벌레 샌드웜 | 기본 이동속도 1.5배 적용 |
| **31~40** | 잊혀진 설원 | 물량형 (개체수 증가) | 서리 거인 | 이속 감소(빙결) 디버프 지속시간 50% 감소 처리 |
| **41~50** | 용암 지대 | 재생형 (초당 회복) | 화염 드래곤 | 매 초마다 최대 HP의 1% 자동 회복 로직 추가 |
| **51~60** | 유령 묘지 | 마법형 (물리 면역) | 리치 킹 | 받는 물리 데미지 90% 감소 |
| **61~70** | 기계 도시 | 속도+체력 밸런스 | 파괴병기 메카 | 현재 HP가 Max HP의 50% 이하가 되면 이동속도 2배 증가 |
| **71~80** | 맹독의 늪 | 면역형 (상태이상 저항) | 맹독 히드라 | HP 0 도달 시 사망하지 않고 HP 20%를 가진 새끼 몬스터 3기로 분열 |
| **81~90** | 폭풍의 계곡 | 회피형 (공격 회피) | 폭풍의 지배자 | 빙결, 기절 등 이동 제어 상태이상 완전 면역 (무시) |
| **91~100**| 마왕의 심연 | 완성형 (모든 능력치 고) | 최종 마왕 | 체력 보정 배율 최대로 적용 |

## 2. 몬스터 스탯 성장 공식 (Balancing Formula)

**[AI Agent 지침]** 하드코딩을 피하고, 아래 수학적 공식을 사용하여 라운드별 스펙을 동적으로 계산(Scale)할 것.

### 2.1. 몬스터 체력 (HP)
* **일반 몬스터 HP:** `FLOOR(100 * (1.15 ^ (currentRound - 1)))`
* **보스 몬스터 HP:** * 10R ~ 90R: `FLOOR(해당 라운드 일반 몬스터 HP * 30)`
  * 100R (최종): `FLOOR(해당 라운드 일반 몬스터 HP * 100)`

### 2.2. 방어력 (Armor) 및 실제 피해량 계산
* **방어력 공식:** `FLOOR(currentRound * 1.5)`
* **실제 데미지 적용 공식:** `MAX(1, FLOOR(AttackPower * (100 / (100 + Armor))))`
  * **중요:** 방어력이 아무리 높아도 최소 1의 데미지는 들어가도록 `MAX` 함수로 엣지 케이스를 방어할 것.

### 2.3. 등장 수량 (Quantity)
* **일반 라운드 몬스터 수량:** `MIN(15 + FLOOR((currentRound - 1) / 10) * 5, 60)`
  * 1~9R: 15기 / 11~19R: 20기 ... 최대 60기 제한.
* **보스 라운드 수량:** 무조건 `1`기.
