# 🎮 GameVault — 친구들과 즐기는 게임 모음

친구들과 모였을 때, 혼자 심심할 때.  
설치 하나로 다양한 게임을 바로 즐기세요.

---

## 어떤 앱인가요?

GameVault는 스마트폰 하나로 여러 가지 미니게임을 즐길 수 있는 게임 모음 앱입니다.  
별도 서버 연결 없이, 같은 자리에 모인 친구들과 함께 플레이하는 **오프라인 파티 게임**부터  
혼자서 두뇌를 굴리는 **솔로 게임**까지 한곳에 담았습니다.

---

## 수록 게임

### 🎉 파티 게임 (여럿이 함께)

| 게임 | 설명 |
|---|---|
| 👁️ **눈치 게임** | 1부터 순서대로 숫자를 외쳐라. 같은 숫자를 동시에 부르면 탈락! |
| 🏰 **타워 디펜스** | 몰려오는 몬스터들로부터 성을 지켜라. 등급별 화려한 캐릭터 스킬! |
| ⚡ **반응속도 대결** | 화면이 바뀌는 순간 가장 빨리 누르는 사람이 이긴다. |
| ⚖️ **밸런스 게임** | 둘 중 하나를 골라라. 다수결로 결과가 갈린다. |
| 🔤 **끝말잇기** | 앞 사람이 말한 단어의 끝 글자로 새 단어를 이어라. 타이머 안에! |
| 🕵️ **스파이 게임** | 우리 중 한 명은 주제를 모른다. 스파이를 찾아내라. |

### 🧠 솔로 게임 (혼자 도전)

| 게임 | 설명 |
|---|---|
| 🔢 **숫자 기억** | 잠깐 보여준 숫자들을 순서대로 기억해서 입력하라. |
| 🎨 **색깔 맞추기** | 글자의 **의미**와 **색상** 중 무엇을 따라야 할지 순간 판단하라. |
| 🔡 **초성 퀴즈** | 초성만 보고 단어를 맞혀라. 생각보다 어렵다. |

## 카테고리 필터

홈 화면 상단에서 원하는 유형만 골라볼 수 있습니다.

| 필터 | 어떤 게임? |
|---|---|
| 전체 | 모든 게임 보기 |
| 파티 | 여럿이 함께 즐기는 게임 |
| 혼자 | 혼자서 도전하는 게임 |
| 반응속도 | 순발력이 필요한 게임 |
| 두뇌 | 생각하고 판단하는 게임 |

## 설치 방법

### Play Store (준비 중)

> Google Play 심사 후 업데이트 예정입니다.

### APK 직접 설치

1. [Releases 페이지](../../releases)에서 최신 `.apk` 파일을 다운로드합니다.
2. 안드로이드 기기에서 **설정 → 보안 → 출처를 알 수 없는 앱 설치** 를 허용합니다.
3. 다운로드한 APK 파일을 열어 설치합니다.

> **요구 사항**: Android 8.0 (API 26) 이상

## 이런 상황에 딱 맞아요

- 친구들과 모였는데 뭘 할지 모를 때

- 두뇌 훈련이나 반응속도를 테스트하고 싶을 때
